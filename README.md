🎓 College Website

A responsive and modern college website built using HTML, CSS, and JavaScript. This project demonstrates front-end development skills by creating a multi-page educational institution website with essential sections like courses, admissions, and contact.

🌐 Live Demo

🔗 https://tirthgediya.github.io/college-website/

📌 Features
🏫 Home page with introduction banner
📖 About section with college details
🎓 Courses / Departments section
📞 Contact page with form
🧭 Navigation bar for smooth browsing
💻 Clean and structured UI layout
🛠️ Technologies Used
HTML5
CSS3
JavaScript

🚀 Getting Started

To run this project locally:

Clone the repository
git clone https://github.com/tirthgediya/college-website.git
Navigate to the project folder
cd college-website
Open index.html in your browser
